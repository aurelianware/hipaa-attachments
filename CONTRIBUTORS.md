## Contribution Types

We recognize various types of contributions:

### Code Contributions
- Feature implementations
- Bug fixes
- Performance improvements
- Refactoring and optimization
- Test coverage improvements

### Documentation Contributions
- Technical documentation
- Tutorials and guides
- API documentation
- Translation and localization
- Video content

### Community Contributions
- Answering questions in GitHub Discussions
- Reviewing pull requests
- Reporting bugs with detailed reproductions
- Suggesting and discussing new features
- Helping other contributors

### Bounty Contributions
- Completing bounty work
- High-value feature implementations
- Complex technical challenges
- Platform enhancements

## How to Be Recognized

### For Code/Documentation Contributors

1. Submit a pull request following [CONTRIBUTING.md](CONTRIBUTING.md)
2. Get your PR approved and merged
3. You'll be automatically added to this list in the next update

### For Bounty Contributors

1. Complete a bounty per [BOUNTY-PROGRAM.md](BOUNTY-PROGRAM.md)
2. Get your bounty PR approved and merged
3. Receive payment
4. You'll be added to the Top Bounty Earners section

### For Community Contributors

Active community members who consistently help others will be recognized quarterly.

## Recognition Benefits

### All Contributors Receive

- Name and GitHub profile in CONTRIBUTORS.md
- Contributor badge for GitHub profile
- Recognition in release notes (for significant contributions)
- Invitation to contributor-only discussions

### Top Contributors Receive

- Cloud Health Office Sentinel swag (t-shirts, stickers)
- Feature highlights on project social media
- Invitation to annual contributor summit (virtual)
- Priority support for their contributions

### Core Contributors Receive

*Earned after 5+ merged PRs or significant contributions:*

- Write access to repository
- Ability to review pull requests
- Voice in project direction and roadmap
- Exclusive recognition and swag
- Early access to new features

## Code of Conduct

All contributors must follow the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**In summary:**
- Be respectful and inclusive
- Welcome newcomers
- Focus on what is best for the community
- Show empathy towards other community members

Violations can be reported to conduct@aurelianware.com.
