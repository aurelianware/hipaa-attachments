Process Flow

  1. SFTP Polling: Monitors Availity SFTP for new 275 attachment files
  2. Data Lake Archive: Stores raw files with date-based partitioning
  3. X12 Decoding: Processes EDI messages via Integration Account
  4. Metadata Extraction: Extracts claim, member, and provider information
  5. Backend Integration: Links attachments to claims via your system's API
  6. Event Publishing: Publishes to Service Bus for downstream processing
  7. 277 RFAI: Generates and sends status responses to Availity
  8. 278 Processing: Handles authorization requests and responses
